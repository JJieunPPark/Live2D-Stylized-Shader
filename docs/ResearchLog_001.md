# Live2D Style Interactive 3D Shader 연구 기록

## 프로젝트 목표

본 프로젝트의 목표는 기존 NPR(Non-Photorealistic Rendering) 기반 툰 셰이더를 발전시켜, 2D 일러스트와 유사한 표현력을 가지면서도 실시간 게임 환경에서 동작 가능한 인터랙티브 3D 렌더링 시스템을 개발하는 것이다.

기존 애니메이션풍 게임들은 캐릭터 모델에 셀 셰이딩(Cell Shading)을 적용하지만, 실제 일러스트레이터가 그린 2D 캐릭터의 명암, 하이라이트, 엔젤링, 머리카락 결 표현까지 재현하는 데에는 한계가 존재한다.

본 연구는 "3D 모델이지만 2D 일러스트처럼 보이는 렌더링", "Live 2D 같은 3D"를 목표로 한다.

참고(Minh Tuan Bui, Junho Kim, Yunjin Lee, 3D-look shading from contours and hatching strokes,Computers & Graphics,
(https://www.sciencedirect.com/science/article/pii/S0097849315000734))
---

# 1. 문제 정의

현재의 툰 셰이더는 대부분 다음 방식으로 동작한다.

* 법선(Normal) 기반 조명 계산
* 명암 단계 분리
* 림 라이트(Rim Light)
* 외곽선(Outline)

그러나 실제 일러스트에서는 다음과 같은 요소들이 존재한다.

* 광원 방향과 무관한 그림자
* 작가가 의도적으로 넣은 하이라이트
* 머리카락의 결 방향을 따라가는 반사광
* 얼굴 부위별로 다른 명암 규칙
* 카메라 구도에 따라 달라지는 표현

즉, 실제 그림은 물리적으로 계산되지 않고 "그렇게 보이도록 그려진 것"에 가깝다.

따라서 기존 PBR 기반 접근만으로는 한계가 존재한다.

---

# 2. 초기 아이디어

초기에는 그림자 처리 방식을 변경하는 아이디어를 고려하였다.

일반적인 렌더링 파이프라인은 다음과 같다.

1. 광원 계산
2. 그림자 계산
3. 최종 색상 출력

하지만 툰 렌더링에서는 오히려

1. 그림자 형태 결정
2. 나머지 영역에 조명 적용

순서가 더 적합할 수 있다고 판단하였다.
특히 애니메이션 스타일에서는 그림자의 형태 자체가 캐릭터 디자인의 일부로 작동하기 때문이다.

---

# 3. 아티스트 제어 방식 연구

인터랙티브한 렌더링을 구현하기 위해 다음 방법을 검토하였다.

## 방법 1 : 수작업 마스크

아티스트가 직접

* 얼굴 그림자
* 머리카락 하이라이트
* 엔젤링
* 특수 음영

등의 마스크를 제작한다.

### 장점

* 높은 품질
* 원하는 결과 보장

### 단점

* 제작 비용 증가
* 캐릭터, 장면마다 별도 작업 필요

---

## 방법 2 : 수식 기반 생성

마스크를 사용하지 않고

* Normal
* Curvature
* Tangent
* View Direction

등을 활용하여 실시간으로 계산한다.

### 장점

* 완전한 인터랙티브 구현 가능
* 추가 텍스처 필요 없음

### 단점

* 일러스트 수준 품질 확보가 어려움

---

# 4. 뉴럴 렌더링 활용 아이디어

수작업 마스크의 제작 비용을 줄이기 위해 뉴럴 렌더링 기반 접근을 고려하였다.

과정은 다음과 같다.

1. 게임 최초 실행 시 학습 데이터 로드
2. 아티스트가 제작한 마스크 분석
3. 마스크를 근사하는 수식 생성
4. 런타임에서는 수식만 사용

즉 비싼 계산은 처음 한 번만 수행하고, 실제 플레이에서는 가벼운 셰이더만 사용하는 구조를 목표로 한다.

### 기대 효과

* 아티스트 의도 유지
* 런타임 성능 확보
* 마스크 의존성 감소

---

# 5. 머리카락 표현 연구

일본 애니메이션 스타일에서 중요한 요소는 머리카락 표현이다.
특히 다음 요소에 주목하였다.

## 엔젤링(Angel Ring)

머리카락 위에 나타나는 띠 형태의 하이라이트.
일반적인 PBR 반사와 다르게 동작한다.

---

## 이방성 반사(Anisotropic Reflection)

머리카락 결 방향을 따라 늘어지는 반사광.
Pencil+4 등 NPR 렌더러에서 자주 사용된다.
(참고: https://cgworld.jp/article/202602-mellow-3.html )

### 연구 방향

* Tangent 방향 활용
* BRDF 수정
* 결 방향 기준 그림자 생성
기존 물리 기반 반사가 아닌 애니메이션 스타일 반사 모델 개발을 목표로 한다.

---

# 6. 시네마틱 툰 셰이더 최적화

교수님 피드백에 따르면
"시네마틱 수준 NPR 효과는 일반 PC에서 실시간 실행하기 어렵다."
는 문제가 존재한다.

따라서 본 연구의 핵심 과제는
"어떻게 하면 고품질 NPR 효과를 일반 게이밍 PC에서도 실시간으로 실행할 수 있는가"
로 정리되었다.

연구 우선순위는 다음과 같다.

1. 시각 품질 유지
2. 계산량 감소
3. 인터랙티브성 확보
4. 범용 GPU 지원

---

# 7. 실시간 적용 범위 논의

프로젝트 초기에는

* 플레이어
* NPC
* 배경

전체에 적용하는 방안을 고려하였다. 이후 성능 문제를 고려하여 우선적으로 캐릭터에만 적용하는 방향으로 수정하였다.

특히
* 플레이어
* 주요 NPC

중심 적용 후, 추후 배경 확장을 검토한다.

---

# 8. 게임 엔진 수준 확장 가능성

게임의 세계관이나 지역이 변경될 때 렌더링 스타일도 함께 변경되는 시스템을 구상하였다.

예시:
* 현실 세계 → 반실사 스타일
* 환상 세계 → 애니 스타일
* 기억 세계 → 수채화 스타일

즉 셰이더 자체를 게임 연출 요소로 사용하는 방향이다. 이는 단순한 그래픽 기술이 아니라 게임 디렉션 기술로 확장될 수 있다.
