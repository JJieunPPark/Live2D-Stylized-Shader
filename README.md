# Live2D-Stylized-Shader
Research project for real-time interactive 2D-stylized rendering on 3D characters.

## Goal

This project explores how to reproduce Live2D-like visual expressions on real-time 3D models.

The main goal is to create an interactive toon shader that can support:

- anime-style lighting
- controllable shadow shapes
- stylized highlights
- dynamic expression changes
- real-time performance

## Motivation

2D character illustrations often use hand-drawn shadows, highlights, and shape exaggeration.
However, 3D real-time rendering usually depends on physically based lighting.

This project investigates how to combine artistic 2D-style control with interactive 3D rendering.
